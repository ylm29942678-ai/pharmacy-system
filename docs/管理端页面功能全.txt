管理端页面功能全
*页面1
控制台页面
页面定位
系统登录后默认工作台，整合核心数据、快捷操作与业务提醒，是日常管理总入口，满足乡镇药房基础运营管控需求。
核心基础功能
1. 用户与导航功能
展示登录用户姓名、角色，提供密码修改、退出登录操作；承载左侧全模块导航栏，实现各数据表页面一键跳转，布局简洁易操作。
2. 核心数据概览
实时展示当日营业额、订单总数、库存预警药品数、近效期药品数、当月采购总金额；同步显示会员总数、活跃会员数，直观掌握药房核心运营数据。
3. 业务快捷操作
设置高频业务快捷按钮：新增销售订单、采购入库、药品信息录入、会员登记、库存盘点，简化业务操作流程。
4. 近期业务记录
展示最新5条销售订单、采购入库记录，显示订单号、时间、金额等关键信息，支持快速查看详情。


迭代开发功能
1. 新增销售、采购数据可视化图表，支持按日/月/季度查看经营趋势
2. 增加库存预警、近效期药品弹窗提醒与消息中心推送
3. 对接收银、打印设备，实现扫码开单与小票快速打印
4. 支持经营数据、业务记录批量导出Excel，便于存档与对账
5. 支持工作台自定义布局，可隐藏、调整模块展示顺序
6. 增加会员生日提醒、消费频次分析等会员运营辅助功能
7. 新增系统运行状态监控，对异常情况进行自动提示


*页面2
药品信息页面
页面定位
药品基础信息管理核心页面，负责中药、西药、医疗器械的全生命周期信息维护，支撑系统销售、库存、采购业务的数据调用。
核心基础功能
1. 药品信息列表展示
以表格形式展示所有药品信息，包含药品名称、类型、规格、剂型、零售价格、生产厂家、是否处方药、库存预警状态等关键字段，支持分页查看。
2. 药品信息查询筛选
支持按药品名称、药品类型（中药/西药/器械）、在售状态精准/模糊查询，快速定位目标药品。
3. 药品信息增删改查
实现药品信息新增、编辑、删除操作；对停售药品采用逻辑删除（通过 status 字段标记为0，数据保留在原表不物理删除），保障数据可追溯。
4. 详情查看
点击单条药品详情按钮，查看完整信息，包括批准文号、参考进价、备注、创建及更新时间等详细内容。


迭代开发功能
1. 新增药品图片上传与展示功能
2. 增加药品禁忌、适用症状等详细信息录入
3. 支持药品信息批量导入、导出Excel
4. 关联库存数据，页面直接查看对应药品各批号库存数量
5. 新增药品价格修改日志记录，便于追溯调价记录


*页面3
供应商页面
页面定位
供应商信息统一管理页面，维护药品供应商基础数据，为采购业务、库存溯源提供数据支撑。
核心基础功能
1. 供应商信息列表展示
表格形式展示所有供应商信息，包含供应商名称、联系人、联系电话、供应类型、合作状态等关键字段，分页展示。
2. 供应商信息查询筛选
支持按供应商名称、简称、供应类型、合作状态模糊/精准查询，快速查找目标供应商。
3. 供应商信息增删改查
完成供应商信息新增、编辑、删除，对暂停合作的供应商进行停用标记，保障采购业务可溯源。
4. 详情查看
通过详情按钮查看供应商完整信息，包括地址、营业执照号、银行信息、备注、创建及更新时间。


迭代开发功能
1. 关联采购数据，查看对应供应商历史采购记录、采购总额
2. 支持供应商信息批量导出，便于线下归档管理
3. 新增供应商信誉评级、供货时效标注
4. 添加供应商联系人信息批量维护功能
5. 对接通讯功能，支持直接拨打供应商联系电话


*页面4
库存页面
页面定位
药品库存核心管理页面，按药品+批号精细管理，实现库存查询、效期监控、药柜定位与出入库追溯。
核心基础功能
1. 库存列表展示
表格展示药品名称、批号、生产日期、有效期、库存数量、成本价、药柜位置、供应商等信息，分页显示。
2. 条件筛选查询
按药品名称、批号、药柜类型、是否近效期、供应商筛选，快速定位库存。
3. 库存明细查看
查看单一批号药品完整信息，支持追溯来源采购单。
4. 库存状态监控
标记近效期、库存预警、过期药品，醒目颜色区分提醒。


迭代开发功能
1. 销售自动扣减、采购自动增加对应批号库存
2. 库存调拨功能，支持不同药柜间移库
3. 近效期、过期药品自动锁定，禁止销售
4. 库存数据导出Excel，支持盘点对账
5. 库存变动日志记录，便于追溯异常


*页面5
用户页面
页面定位
系统操作员（管理员/店员）账号管理页面，用于账号新增、权限分配、信息维护，保障系统操作安全与权责区分。
核心基础功能
1. 用户信息列表展示
以表格展示账号、姓名、角色（管理员/店员）、联系电话、状态等信息，分页显示。
2. 用户查询筛选
支持按用户名、姓名、角色、状态进行快速筛选。
3. 用户信息增删改查
实现用户新增、编辑、删除，可重置密码、启用/禁用账号；对停用账号采用逻辑删除机制，通过status字段标记为0，数据保留在原表不物理删除，保障操作记录可追溯。
4. 权限与状态管理
系统预设管理员、店员两种基础角色，精准划分不同角色的模块操作权限；前端根据权限动态渲染功能按钮，后端接口同步校验，双重保障防止越权操作；通过status字段管控账号启用/禁用状态，实现账号生命周期管理。


迭代开发功能
1. 记录账号登录日志，展示登录时间、IP与设备信息
2. 密码强度校验、定时自动修改提醒
3. 操作权限精细化配置，支持自定义角色与按钮级权限控制
4. 用户工作量统计，关联销售、采购、盘点记录
5. 批量导入导出员工账号信息


*页面6
顾客页面
页面定位
顾客及会员信息管理页面，用于登记、查询、维护顾客资料，管理会员等级与消费记录，支撑销售结算时的会员识别与优惠。
核心基础功能
1. 顾客信息列表展示
以表格展示顾客姓名、电话、是否会员、会员等级、累计消费金额、状态等信息，分页显示。
2. 顾客信息查询筛选
支持按顾客姓名、手机号、会员等级、是否会员进行模糊或精确筛选。
3. 顾客信息增删改查
完成顾客信息新增、编辑、删除，支持将会员启用/禁用，维护会员等级。
4. 顾客详情查看
查看顾客完整资料、累计消费、会员信息及备注。


迭代开发功能
1. 关联订单数据，查看该顾客历史消费订单与消费明细
2. 支持会员积分管理、积分抵扣设置
3. 顾客信息批量导入导出Excel
4. 生日自动提醒、会员充值与储值消费功能
5. 会员等级自动升级规则配置
6. 支持顾客手机号+密码登录用户端，实现会员自助查询、订单查看等功能


*页面7
订单主表页面
页面定位
销售订单核心管理页面，集中展示所有销售订单整体信息，实现订单查询、状态管理、数据统计，为销售结算、账务核对提供数据支撑。
核心基础功能
1. 订单列表展示
以表格形式展示订单号、顾客信息、操作员、下单时间、订单总价、支付方式、支付状态等关键字段，分页展示，排序清晰。
2. 订单多条件筛选查询
支持按订单号、下单时间、顾客手机号、支付方式、支付状态精准/模糊查询，可按日/月筛选订单。
3. 订单详情查看
点击单条订单行，通过 URL 携带 order_id 参数跳转到订单明细表页面，自动加载并展示该订单下的所有药品明细。
4. 订单状态管理
支持查看订单支付状态，对已支付、已退款订单进行状态标记，可添加订单备注信息。


迭代开发功能
1. 订单数据统计，支持按时间段统计销售额、订单量，生成简易统计数据
2. 订单小票打印、订单信息导出 Excel 功能
3. 支持订单退款、作废操作，同步回滚库存数据
4. 关联顾客累计消费，自动更新会员消费总额
5. 新增订单批量查询、批量导出功能


*页面8
订单明细表页面
页面定位
展示单笔销售订单的药品明细信息，用于核对销售内容、追溯药品批号与库存来源，是订单主表的详情配套页面。
核心基础功能
1. 订单明细列表展示
以表格形式展示对应订单的药品名称、批号、单价、销售数量、小计金额等信息，清晰展示每一笔药品销售明细。
2. 根据订单号自动筛选
通过 URL 接收 order_id 参数，页面加载时自动查询并仅显示当前订单的明细数据，无需手动筛选。
3. 关联信息查看
展示关联的订单主表信息（订单号、操作员、下单时间、总金额），方便对账核对。
4. 明细数据查阅
支持查看订单内完整药品销售明细，逐条展示药品名称、批号、数量、单价、金额等信息，可追溯药品来源与批次，便于效期核对、库存核对与销售对账。


迭代开发功能
1. 支持订单明细导出、打印小票
2. 退款时可选择明细药品单独退款，同步回滚对应批号库存
3. 展示药品零售价与成本价，计算单笔订单毛利
4. 关联顾客信息，支持查看该顾客历史同类药品购买记录
5. 增加异常订单明细标记与备注功能


*页面9
采购主表页面
页面定位
采购业务核心管理页面，统一管理所有采购入库单据，用于统计各供应商进货金额、对账与采购溯源，是采购业务的总览入口。
核心基础功能
1. 采购单列表展示
以表格展示采购单号、供应商、操作员、采购时间、总金额、支付方式、入库状态等信息，分页展示。
2. 多条件筛选查询
支持按采购单号、供应商、时间段、入库状态筛选，方便按月对账、按供应商统计进货数据。
3. 采购单详情查看
点击单条采购单，通过 URL 携带 purchase_id 参数跳转到采购明细表页面，自动加载该单据下所有药品明细。
4. 采购状态管理
标记待入库、已入库、作废状态，支持编辑备注，方便跟踪采购到货与入库情况。


迭代开发功能
1. 按供应商/时间段统计采购总额、进货量，生成简单汇总数据
2. 采购单导出 Excel、打印采购单据功能
3. 支持采购单作废、部分入库操作
4. 自动对比历史采购价，提示价格异常
5. 批量导出采购数据，用于财务对账


*页面10
采购明细表页面
页面定位
展示单笔采购单的药品明细信息，记录本次入库药品的名称、数量、进价、批号、效期与存放药柜，作为库存入库的直接依据，与采购主表联动展示。
核心基础功能
1. 采购明细列表展示
以表格展示药品名称、批号、生产日期、有效期、采购单价、采购数量、小计金额、建议存放药柜等信息，清晰呈现每味药品采购详情。
2. 根据采购单号自动筛选
通过 URL 接收 purchase_id 参数，页面加载后自动筛选并只显示当前采购单的明细数据。
3. 关联采购单信息展示
顶部显示对应采购主表信息：采购单号、供应商、操作员、采购时间、总金额，便于核对与溯源。
4. 库存入库依据查阅
完整保留批号、效期、进价等核心字段，作为库存新增与财务成本核算的依据。


迭代开发功能
1. 支持采购明细导出、打印入库单
2. 入库审核功能，审核通过后自动同步增加对应药品库存
3. 显示历史采购价，对比本次进价是否异常
4. 按药品统计该供应商历史供货价格趋势
5. 支持批量修改药柜位置、备注等信息


*页面11
盘点表页面
页面定位
药品库存盘点专用页面，用于记录账面库存与实际盘点数量，自动计算盈亏数据，实现药房定期盘点、账实核对与损耗管理。
核心基础功能
1. 盘点记录列表展示
以表格展示盘点单号、药品名称、批号、账面数量、实盘数量、盈亏数量、盘点人、盘点时间等信息，分页展示。
2. 多条件筛选查询
支持按盘点时间、药品名称、批号、盘点人筛选，可按单次盘点编号统一查看本次盘点全量数据。
3. 盘点信息新增与录入
选择对应药品及批号，自动从库存表stock的stock_num字段获取账面库存，工作人员现场清点后录入实际库存，系统依据“实际库存-账面库存”自动计算盈亏数量，结合药品成本价自动算出盈亏金额。
4. 盈亏原因记录
对盘盈、盘亏、破损、过期等情况填写原因说明，留存盘点备注，便于后续财务核查与责任追溯。


迭代开发功能
1. 支持盘点数据导出 Excel，用于存档与对账
2. 盘点确认后自动调整系统库存，生成库存变动记录
3. 按时间段统计盘点损耗总金额，形成盘点报表
4. 批量盘点导入功能，支持扫码快速录入实盘数量
5. 盘点差异过大时自动高亮提醒，减少人为出错


*页面12
日志表页面
页面定位
系统操作审计与追溯页面，全程记录所有用户操作行为，用于操作核查、问题排查、权责界定，保障系统数据安全与业务可追溯。
核心基础功能
1. 操作日志列表展示
以表格展示日志ID、操作人姓名、操作模块、操作类型、操作内容、操作IP、操作时间，分页展示所有操作记录。
2. 多条件筛选查询
支持按操作人、操作模块（销售/采购/库存等）、操作类型、操作时间段精准筛选，快速定位目标操作日志。
3. 操作详情查看
点击单条日志，查看完整操作内容、操作终端IP等详细信息，清晰还原操作场景。
4. 日志权限管控
仅管理员可查看全量日志，普通店员无日志页面访问权限，保障操作记录保密性。


迭代开发功能
1. 日志数据定时归档，支持导出Excel存档
2. 新增异常操作（违规删除、越权操作）自动标记预警
3. 按操作模块/操作人统计操作频次，生成操作统计报表
4. 日志数据加密存储，防止篡改，提升数据安全性
